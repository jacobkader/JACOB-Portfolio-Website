Runner

My Role

To program a fully functioning animated game using Java(processing) focusing primaly on player moving using key pads, enemies, collectables, score sheet and anything extra up to preference.

Project Difficulties

This project really tested my skills in working with several classes. In addition I found it challanging to make the code sufficient with there being individual classes for each collectable,enemy and character.

My Solution

To solve this struggle I learned about a few techniques called Polymorphism and inheritance, together I combined them to parental classes depending on data type. Apples, Hearts and golden coins would be classes as Collectables and hence operate through the parentall class collectables. The same was done for Enemies and Characters.

Technologies used

-Java (Processing)
